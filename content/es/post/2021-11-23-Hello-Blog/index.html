Desde que me di cuenta de todas las posibilidades sobre la codificación en R -en su mayoría- pero también en otro lenguaje, que quería crear un blog para compartir algunas de las cosas que he aprendido. Sin embargo, me gusta también la idea de un blog como una plataforma para compartir el conocimiento entrante como un proceso de aprendizaje-enseñanza. Así que ahora, al final de mi doctorado, parece ser el momento y la oportunidad adecuados para hacerlo, al tiempo que se crea un portafolio de las cosas y habilidades que he recogido en el proceso. La mayoría de estas habilidades son difíciles de compartir a través de publicaciones científicas, pero pueden ser útiles para otros en el mismo viaje de aprendizaje

Para demostrar la utilidad de aprender un lenguaje de programación como herramienta para el desarrollo profesional, y para demostrar que no se necesita un título en informática para hacerlo, decidí crear este blog con la ayuda de {Rmarkdown}{blogdown}, HUGO, Ananke Theme del grupo New Dynamic para crear el blog, junto con GitHub y Netifly para el despliegue del sitio. Esto también requirió el uso de algo de CSS, y HTML para modificar el tema a mi preferencia, y algo de código en R y Git para crear y desplegar la página. Por cierto, ¿He mencionado que todas las herramientas de codificación y el lenguaje que uso son de código abierto y/o tienen planes gratuitos como Netifly y GitHub? Además también tienen una enorme comunidad como #RStat donde puedes encontrar ayuda y consejos. Así que sí, salvo el dominio todo lo demás en el despliegue es gratuito, una razón más para empezar a aprender hoy, y utilizar tus habilidades y conocimientos como un proyecto de futuro.

¡Así que espero que disfrutes de tu visita y aprendas algo nuevo!
